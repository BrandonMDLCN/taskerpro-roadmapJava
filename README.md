<h1>Tasker Pro</h1>
<h2>Función Generación y manejo de Tareas</h2>
<p>Sencilla aplicación CLI en cual te permite gestionar tus tareas</p>
<p>Todas las tareas se guardan en un archivo JSON permitiendo Exportar/Compartir tus tareas!</p>
Funcionalidades:
1: Crear Tareas Nuevas
2: Visualizar tareas (Se pueden filtrar depende del estatus)
3: Actualizar tarea
4: Eliminar tarea

<h3>Para importar tareas es necesario guardar el archivo JSON en tu Carpeta de Usuario</h3>
